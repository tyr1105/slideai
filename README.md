# SlideAI - Free Online PPT Generator

Live at: https://tyr1105.github.io/slideai/

## Features
- 5 professional templates (Business, Pitch, Education, Creative, Minimal)
- Auto-generate slide outlines from a topic
- Live preview with slide editor
- Export to PPTX (compatible with WPS and Office)
- 100% client-side, no signup required
- Mobile responsive

## Tech Stack
- React + Vite
- Tailwind CSS
- PptxGenJS for PPTX export
- Deployed on GitHub Pages
